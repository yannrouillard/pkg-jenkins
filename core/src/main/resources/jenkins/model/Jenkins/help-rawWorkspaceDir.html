Specify where Jenkins would store job workspaces on the master node. This value can include the following variables.
  • ${JENKINS_HOME} — Jenkins home directory
  • ${ITEM_ROOTDIR} — Root directory of a job for which the workspace is allocated.
  • ${ITEM_FULLNAME} — '/'-separated job name, like "foo/bar".

Changing this value allows you to put workspaces on SSD, SCSI, or even ram disks. Default value is ${ITEM_ROOTDIR}/workspace.
